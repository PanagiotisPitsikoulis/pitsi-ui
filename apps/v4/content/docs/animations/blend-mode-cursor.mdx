## Props

### BlendModeCursor

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `30` | Size of the cursor when not active |
| `activeSize` | `number` | `400` | Size of the cursor when active/hovering |
| `color` | `string` | `"#BCE4F2"` | Cursor color |
| `activeBlur` | `number` | `30` | Blur amount when active |
| `stiffness` | `number` | `150` | Spring stiffness for cursor movement |
| `damping` | `number` | `15` | Spring damping for cursor movement |

### GradientCursor

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `30` | Size of each circle when not active |
| `activeSize` | `number` | `400` | Size of each circle when active/hovering |
| `colors` | `string[]` | `["#c32d27", "#f5c63f", "#457ec4", "#356fdb"]` | Array of colors for the gradient circles |
| `activeBlur` | `number` | `20` | Blur amount when active |
| `blur` | `number` | `2` | Blur amount when not active |

Add `data-cursor-active` attribute to any element to trigger the active state on hover.
