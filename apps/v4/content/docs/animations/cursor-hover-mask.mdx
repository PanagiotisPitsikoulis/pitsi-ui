## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maskContent` | `ReactNode` | - | Content shown in the mask (revealed content) |
| `children` | `ReactNode` | - | Content shown in the background (base content) |
| `size` | `number` | `40` | Size of the mask when not hovering |
| `hoverSize` | `number` | `400` | Size of the mask when hovering |
| `maskBackground` | `string` | `"#ec4e39"` | Background color of the mask area |
