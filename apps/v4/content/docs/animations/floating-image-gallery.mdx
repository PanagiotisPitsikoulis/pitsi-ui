## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `layers` | `Array<{ images, speed?, brightness? }>` | - | Array of image layers. Each layer moves at different speed |
| `children` | `ReactNode` | - | Content to display in the center |
| `speed` | `number` | `0.01` | Speed multiplier for mouse movement |
| `easing` | `number` | `0.08` | Easing factor for animation smoothness |
