## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `projects` | `Array<{ title, src, color, subtitle? }>` | - | Array of projects to display |

### Project Object

| Property | Type | Description |
|----------|------|-------------|
| `title` | `string` | Project title |
| `src` | `string` | Image source URL |
| `color` | `string` | Background color for the image modal |
| `subtitle` | `string` | Optional subtitle (default: "Design & Development") |
