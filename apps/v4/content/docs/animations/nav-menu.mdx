## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `brand` | `ReactNode` | `"Brand"` | Brand name or logo content |
| `links` | `Array<{ title, href, src? }>` | - | Array of navigation links |
| `rightContent` | `ReactNode` | - | Right side content (e.g., cart, CTA) |

### NavLink Object

| Property | Type | Description |
|----------|------|-------------|
| `title` | `string` | Link title |
| `href` | `string` | Link URL |
| `src` | `string` | Optional image URL to show on hover |
