---
title: TanStack Start
description: Install and configure pitsi/ui for TanStack Start.
---

<Steps>

### Create project

Run the following command to create a new TanStack Start project with pitsi/ui:

```bash
npm create @tanstack/start@latest --tailwind --add-ons pitsi
```

### Add Components

You can now start adding components to your project.

```bash
npx pitsi@latest add button
```

The command above will add the `Button` component to your project. You can then import it like this:

```tsx title="app/routes/index.tsx" showLineNumbers {1,6}
import { Button } from "@/components/ui/button"

function App() {
  return (
    <div>
      <Button>Click me</Button>
    </div>
  )
}
```

</Steps>

If you want to add all `pitsi/ui` components, you can run the following command:

```bash
npx pitsi@latest add --all
```
