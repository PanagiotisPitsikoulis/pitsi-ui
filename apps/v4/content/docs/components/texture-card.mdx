## Usage

```tsx
import {
  TextureCard,
  TextureCardHeader,
  TextureCardTitle,
  TextureCardDescription,
  TextureCardContent,
  TextureCardFooter,
  TextureSeparator,
} from "@/components/ui/texture-card"

function Example() {
  return (
    <TextureCard>
      <TextureCardHeader>
        <TextureCardTitle>Card Title</TextureCardTitle>
        <TextureCardDescription>Card description</TextureCardDescription>
      </TextureCardHeader>
      <TextureSeparator />
      <TextureCardContent>
        Card content goes here
      </TextureCardContent>
      <TextureSeparator />
      <TextureCardFooter>
        <button>Action</button>
      </TextureCardFooter>
    </TextureCard>
  )
}
```

## Components

### TextureCard

The main card container with layered border styling.

### TextureCardHeader

Container for the card header, typically contains title and description.

### TextureCardTitle

Styled heading for the card title.

### TextureCardDescription

Styled paragraph for the card description.

### TextureCardContent

Container for the main card content with appropriate padding.

### TextureCardFooter

Container for card actions, typically buttons.

### TextureSeparator

A styled separator that matches the card's textured aesthetic.
