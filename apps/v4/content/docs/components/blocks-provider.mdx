## Usage

Wrap your blocks with the provider to apply theme styling:

```tsx
import { BlocksProvider } from "@/components/ui/blocks-provider"
import { Hero1 } from "@/components/blocks/hero-1"

export default function Page() {
  return (
    <BlocksProvider slug="service-plants" tint="tinted">
      <Hero1 />
    </BlocksProvider>
  )
}
```

## Props

| Prop        | Type                         | Default   | Description                                               |
| ----------- | ---------------------------- | --------- | --------------------------------------------------------- |
| slug        | string                       | "default" | Template theme slug (e.g., "service-plants", "ai", "art") |
| tint        | "base" \| "tinted" \| "deep" | "base"    | Color intensity level                                     |
| forceDark   | boolean                      | false     | Force dark mode regardless of system preference           |
| forceLight  | boolean                      | false     | Force light mode regardless of system preference          |
| transparent | boolean                      | false     | Make background transparent                               |
| className   | string                       | -         | Additional CSS classes                                    |

## Template Slugs

Available template slugs and their color palettes:

| Category          | Slugs                                                  | Palette             |
| ----------------- | ------------------------------------------------------ | ------------------- |
| AI & Technology   | `ai`, `ai-sci-fi`, `product-scifi`                     | azure, indigo, cyan |
| Creative & Art    | `art`, `service-fashion`, `service-tattoo`             | violet, rose, coral |
| Food & Beverage   | `food-pizza`, `food-juice`, `service-coffee-shop`      | coral, amber        |
| Products          | `product-coffee`, `product-plants`, `product-skincare` | amber, sage, rose   |
| Health & Wellness | `service-gym`, `service-psychologist`                  | coral, sage         |
| Beauty            | `service-barber`, `service-makeup`                     | amber, rose         |
| Lifestyle         | `boat`, `service-travel`, `service-hospitality`        | azure, amber        |
| Professional      | `service-real-estate`, `service-marketing`             | indigo, violet      |

## Tint Levels

- **base**: Neutral colors with no brand tinting
- **tinted**: Subtle brand color accents
- **deep**: Saturated background colors with strong brand presence

## Decoration Components

The provider also exports decoration components for building custom blocks:

### FloatingDecoration

Animated floating shapes for visual interest:

```tsx
import { FloatingDecoration } from "@/components/ui/blocks-provider"

;<FloatingDecoration
  shape="circle"
  size="md"
  position="top-right"
  color="bg-brand/20"
  blur="2xl"
  animate
/>
```

### PatternDecoration

Background patterns:

```tsx
import { PatternDecoration } from "@/components/ui/blocks-provider"

;<PatternDecoration pattern="dots" size="md" opacity={10} fade="radial" />
```

### Overlay

Dark, light, or gradient overlays:

```tsx
import { Overlay } from "@/components/ui/blocks-provider"

;<Overlay variant="gradient-bottom" opacity={60} />
```

### BgDecoration

Background gradient decorations:

```tsx
import { BgDecoration } from "@/components/ui/blocks-provider"

;<BgDecoration variant="glow" opacity={30} position="top" />
```

### KnockoutText

Makes text readable over any background:

```tsx
import { KnockoutText } from "@/components/ui/blocks-provider"

;<KnockoutText blur="3xl" intensity={2}>
  <h1>Readable Text</h1>
</KnockoutText>
```

## Context Components

### BlockContainer

Container with alternating backgrounds:

```tsx
import { BlockContainer } from "@/components/ui/blocks-provider"

<BlockContainer index={0}>
  {/* Even index = page background */}
</BlockContainer>

<BlockContainer index={1}>
  {/* Odd index = muted background */}
</BlockContainer>
```

### useBlockContext

Access block context for styling:

```tsx
import { useBlockContext } from "@/components/ui/blocks-provider"

function MyComponent() {
  const { cardBg, surfaceBg, isEven } = useBlockContext()

  return <div className={cardBg}>...</div>
}
```
