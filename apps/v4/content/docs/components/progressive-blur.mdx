## Usage

```tsx
import { ProgressiveBlur } from "@/components/ui/progressive-blur"
```

```tsx
<div className="relative">
  <img src="/hero-image.jpg" alt="Hero" />
  <ProgressiveBlur
    direction="top"
    blurIntensity={16}
    className="absolute bottom-0 h-1/3 w-full"
  />
</div>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `direction` | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` | The direction of the blur gradient |
| `blurIntensity` | `number` | `8` | The maximum blur intensity in pixels |
| `blurLayers` | `number` | `8` | The number of blur layers |
| `className` | `string` | - | Additional CSS classes |

## Examples

### Bottom Fade

```tsx
<ProgressiveBlur direction="bottom" blurIntensity={12} />
```

### Top Fade

```tsx
<ProgressiveBlur direction="top" blurIntensity={16} />
```

### Horizontal Blur

```tsx
<ProgressiveBlur direction="left" blurIntensity={8} />
```
