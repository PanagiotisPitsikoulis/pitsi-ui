## Usage

```tsx
import { ButtonNeobrutalist } from "@/components/ui/button-neobrutalist"
```

```tsx
<ButtonNeobrutalist variant="default">Click me</ButtonNeobrutalist>
```

## Features

- Hard drop shadows that grow on hover
- Bold 2px borders for a striking look
- Press animation that moves the button down and right
- No rounded corners for a raw, brutalist aesthetic
- Uppercase, bold text with letter spacing

## Examples

### Variants

```tsx
<ButtonNeobrutalist variant="default">Default</ButtonNeobrutalist>
<ButtonNeobrutalist variant="secondary">Secondary</ButtonNeobrutalist>
<ButtonNeobrutalist variant="destructive">Destructive</ButtonNeobrutalist>
<ButtonNeobrutalist variant="outline">Outline</ButtonNeobrutalist>
<ButtonNeobrutalist variant="ghost">Ghost</ButtonNeobrutalist>
```

### Sizes

```tsx
<ButtonNeobrutalist size="sm">Small</ButtonNeobrutalist>
<ButtonNeobrutalist size="default">Default</ButtonNeobrutalist>
<ButtonNeobrutalist size="lg">Large</ButtonNeobrutalist>
```

### Icon

```tsx
<ButtonNeobrutalist size="icon">
  <ArrowRightIcon />
</ButtonNeobrutalist>
```

### With Link

Use the `asChild` prop to render as a link.

```tsx
import Link from "next/link"

<ButtonNeobrutalist asChild>
  <Link href="/docs">Get Started</Link>
</ButtonNeobrutalist>
```

## API Reference

### ButtonNeobrutalist

| Prop      | Type                                                                     | Default     |
| --------- | ------------------------------------------------------------------------ | ----------- |
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost"`      | `"default"` |
| `size`    | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`          | `"default"` |
| `asChild` | `boolean`                                                                | `false`     |
