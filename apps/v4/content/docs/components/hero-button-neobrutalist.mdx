
```tsx

```

## Features

- Hard drop shadows that grow on hover
- Bold 2px borders for a striking look
- Press animation that moves the button down and right
- Integrated arrow icon with slide animation
- Optional expand circle mode for more dramatic hover effect
- Uppercase, bold text with letter spacing

## Examples

### Default

```tsx
<HeroButtonNeobrutalist>Get Started</HeroButtonNeobrutalist>
```

### Expand Circle

Use the `expandCircle` prop for a more dramatic hover animation.

```tsx
<HeroButtonNeobrutalist expandCircle>Explore Now</HeroButtonNeobrutalist>
```

### Variants

```tsx
<HeroButtonNeobrutalist variant="default">Default</HeroButtonNeobrutalist>
<HeroButtonNeobrutalist variant="secondary">Secondary</HeroButtonNeobrutalist>
<HeroButtonNeobrutalist variant="outline">Outline</HeroButtonNeobrutalist>
```

### Sizes

```tsx
<HeroButtonNeobrutalist size="sm">Small</HeroButtonNeobrutalist>
<HeroButtonNeobrutalist size="default">Default</HeroButtonNeobrutalist>
<HeroButtonNeobrutalist size="lg">Large</HeroButtonNeobrutalist>
```

## API Reference

### HeroButtonNeobrutalist

| Prop           | Type                                                                | Default     |
| -------------- | ------------------------------------------------------------------- | ----------- |
| `variant`      | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost"` | `"default"` |
| `size`         | `"default" \| "sm" \| "lg"`                                         | `"lg"`      |
| `expandCircle` | `boolean`                                                           | `false`     |
